# 🏦 Bank Transaction & Fraud Detection Dashboard  

## 🚀 Project Summary  
Financial fraud is a growing challenge in digital banking. This project presents an interactive Power BI dashboard built to analyze transaction patterns, detect fraudulent activities, and generate actionable insights for decision-making.

The dashboard transforms raw transaction data into a clear, visual tool for monitoring performance and identifying risks.

---

## 🎯 Business Problem  
Banks handle large volumes of transactions daily, making it difficult to:  
- Detect fraudulent transactions efficiently  
- Identify high-risk patterns across regions and networks  
- Monitor transaction performance in real time  

This project provides a centralized analytics solution to address these challenges.

---

## 💡 Solution  
Developed a multi-page Power BI dashboard that:  
- Tracks total transactions and transaction value  
- Highlights fraudulent activities using clear indicators  
- Analyzes transaction types, status, and regions  
- Identifies high-risk network slices and bandwidth groups  

---

## 📊 Key Insights  
- Approximately *48% of transactions were flagged as fraudulent*  
- High bandwidth usage shows a strong correlation with fraud  
- Certain regions and network slices exhibit higher fraud concentration  
- Transaction success and failure trends highlight operational gaps
